# Zonghan Du Academic Homepage

Static academic homepage for PhD applications, built as a plain HTML/CSS/JS
site for direct deployment on GitHub Pages.

This version is aligned to the latest CV content. Website sections include
research interests, a PhD-facing research plan, complete research experience,
publications, honors, skills, and the athletics profile requested for the site.

## Files

- `index.html`: homepage structure and academic content
- `styles.css`: visual design, layout, and responsive behavior
- `script.js`: reveal animation, stat count-up, and active navigation
- `.nojekyll`: disables Jekyll processing on GitHub Pages

## Local Preview

Open `index.html` directly, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

For the user site repository `gwkuqgfkqe.github.io`, publish the files in this
folder to the repository root.

Final URL:

